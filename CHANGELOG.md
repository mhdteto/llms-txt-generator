# Changelog

## v1.0 - Initial Public Release

Released: 2026-06-14

### Added

- Basic Python generator for `llms.txt`.
- Reusable template.
- Consultant example.
- SaaS example.
- Local business example.
- Sample generated output.
- MIT license.
- Roadmap.
- Changelog.
- Documentation explaining what `llms.txt` is.
- Usage guide for the generator.
- Entity fields guide.
- Output review checklist.

### Purpose

This first version provides a simple and practical way to create an `llms.txt` file for AI visibility, entity documentation and content review workflows.

### Notes

This project does not guarantee rankings, citations or recommendations in AI-generated answers.

It helps structure important public information about a brand, business, consultant, SaaS product or local service provider.
