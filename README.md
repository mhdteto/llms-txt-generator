# llms.txt Generator

A simple template and Python generator for creating `llms.txt` files for consultants, SaaS companies and local businesses.

## Why this matters

AI systems increasingly rely on clear, structured and source-friendly content to understand brands, services and entities.

An `llms.txt` file can help organize key information about a business in a simple, readable format. It is not a ranking guarantee, but it is a useful way to make important entity information easier to inspect and reuse.

## Use case

Use this project when you want to create a clear source file that summarizes a company, its services, target audience and official links for AI visibility work.

## Example

The generator creates a file like:

```txt
# Example Company

Example Company is an AI consulting business based in Casablanca, Morocco.

## Services

- AI audit
- Business automation
- LLM visibility
```

## How to use

1. Edit the business information in `generator.py`.
2. Run the script with Python 3.
3. Review the generated `llms.txt`.
4. Upload it to the root of your website.
5. Link it from relevant documentation or service pages.

## Business value

The output gives consultants and SMBs a structured way to describe their entity, services and positioning for AI visibility audits, content reviews and website documentation.

## Disclaimer

This project does not guarantee rankings, citations or recommendations in AI-generated answers.

It provides a practical way to structure public business information for AI visibility reviews, entity documentation and content audits.
