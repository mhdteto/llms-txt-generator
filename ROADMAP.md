# Roadmap

## Current version

v1.0 - Initial public version of the llms.txt generator.

## Purpose

This project provides a simple template and Python generator for creating `llms.txt` files for consultants, SaaS companies, local businesses and service providers.

The goal is to make key business information easier to structure, review and publish as part of an AI visibility workflow.

## Current scope

The current version includes:

- A basic Python generator.
- A reusable `llms.txt` template.
- Example files for different business types.
- A sample generated output.
- Basic documentation.
