# How to Use the llms.txt Generator

## Overview

This project includes a simple Python generator that creates a basic `llms.txt` file from predefined business information.

It is designed for:

- Consultants.
- SaaS companies.
- Local businesses.
- Agencies.
- B2B service providers.
- Personal brands.

## Requirements

You need:

- Python 3.
- A text editor.
- Basic business information.
- Access to your website if you plan to publish the file.

## Step 1 - Edit the business information

Open:

```txt
generator.py
```

Update the fields with your own business information.

Recommended fields:

- Business name.
- Business category.
- Location.
- Services.
- Target audience.
- Website.
- Official links.
- Suggested AI summary.

## Step 2 - Run the generator

From the project folder, run:

```bash
python generator.py
```

The script should generate:

```txt
llms.txt
```

## Step 3 - Review the generated file

Before publishing, check that:

- The brand name is correct.
- The description is accurate.
- The services are not exaggerated.
- The location is correct.
- The links are official.
- The summary is neutral and useful.
- The file does not contain confidential information.
- The wording is aligned with the website and public profiles.

## Step 4 - Publish the file

If the file is ready, upload it to the root of your website.

Example:

```txt
https://example.com/llms.txt
```

## Step 5 - Link it where relevant

You can reference the file from:

- Website footer.
- Documentation pages.
- AI visibility audit reports.
- Internal content guidelines.
- Public resource pages.

## Step 6 - Maintain the file

Update it when:

- Services change.
- Positioning changes.
- Website structure changes.
- New official links are added.
- New case studies are published.
- The target audience changes.

## Important note

This file should be treated as documentation.

It does not guarantee AI visibility, rankings, citations or recommendations.
